## Introduction

ShipFree is a production-ready Next.js SaaS boilerplate that helps you launch your startup in days, not weeks. In this guide, we'll walk you through everything you need to get started.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** 18 or higher
- **Bun** (package manager and runtime)
- **PostgreSQL** database

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/revokslab/shipfree.git my-saas
cd my-saas
bun install
```

## Configuration

Copy the environment variables file and configure it:

```bash
cp .env.example .env
```

Then edit `.env` with your configuration. At a minimum, you'll need:

- `NEXT_PUBLIC_APP_URL` — your app's base URL
- `DATABASE_URL` — your PostgreSQL connection string

## Running the Development Server

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) and you should see your ShipFree app running.

## What's Included

ShipFree comes with everything you need to build a modern SaaS:

- **Authentication** via Better-Auth with social login support
- **Payments** supporting Stripe, Polar, and LemonSqueezy
- **Email** with multiple provider support
- **Database** with Drizzle ORM and PostgreSQL
- **UI components** built with TailwindCSS and BaseUI
- **SEO** optimization with JSON-LD structured data
